Python 일정

Python은 고급 프로그래밍 언어로, 간단하고 가독성이 높은 문법을 가지고 있다.
Python은 다양한 라이브러리와 프레임워크를 제공하여 데이터 분석, 웹 개발, 인공지능 등 다양한 분야에서 활용된다.

과목 개요

  • 과목명: Python
  • 분야: 프로그래밍
  • 핵심 주제: 문법, 함수, 클래스, 모듈
  • 목표: Python 프로그래밍 기초를 이해하고 실습한다.
  • 교재: 파이썬 코딩 도장

학습 목표

  1. Python의 기본 문법을 이해한다.
  2. 함수와 클래스를 정의하고 사용한다.
  3. 모듈을 import하고 사용한다.
  4. 파일 입출력을 처리한다.
